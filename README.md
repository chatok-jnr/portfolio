# Portfolio (Vite + React)

This is a minimal starter React project using Vite.

Quick start:

```bash
cd portFolio
npm install
npm run dev
```

- Development server: http://localhost:5173 (default)
- Build: `npm run build`
- Preview production build: `npm run preview`

Next steps:
- Replace `src/App.jsx` with your content
- Add routing with `react-router` if needed
- Add deploy workflow (Netlify/Vercel/GitHub Pages)
